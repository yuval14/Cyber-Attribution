# Code of Conduct

## Expected behavior

Participants are expected to:

1. Be respectful and professional.
2. Focus on defensive, educational, and research-oriented cybersecurity work.
3. Support evidence-based discussion.
4. Cite sources where appropriate.
5. Avoid harassment, personal attacks, or discriminatory language.
6. Avoid publishing sensitive, private, or harmful operational details.

## Unacceptable behavior

Unacceptable behavior includes:

1. Harassment or abuse.
2. Discriminatory language or conduct.
3. Sharing private information without permission.
4. Submitting content that is primarily harmful or offensive in purpose.
5. Repeatedly disrupting professional discussion.

## Enforcement

Repository maintainers may remove content, close issues, reject pull requests, or restrict participation when behavior is inconsistent with this code of conduct.
